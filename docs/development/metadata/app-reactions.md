# app > reactions

Path: metadata > app > reactinos.

*As of v9.0.*

## list

A list of reactinos.

*Record[]*

Example:

```json
{
    "list": [
        {
            "type": "Smile",
            "iconClass": "far fa-face-smile"
        }
    ]
}
```

### type

A reaction type. Must not be longer than 10 characters.

### iconClass

An icon CSS class.
